Ayuda para Técnicos

Esta guía es para el personal técnico de Mantenimiento: cómo ver tus asignaciones, iniciar el trabajo, resolver tickets, registrar materiales del almacén y comentar internamente con el dispatcher.

{% if help_all or 'maint.tickets.page.list' in user_perms %}

1. Mis tickets asignados

Entra al listado de Tickets. Filtra por estado Asignado o En Progreso para ver únicamente tus tickets activos.

{% endif %} {% if help_all or 'maint.tickets.api.resolve' in user_perms %}

2. Iniciar progreso

Cuando vas a empezar un trabajo, abre el ticket y pulsa Iniciar Progreso. Esto cambia el estado de Asignado → En Progreso y avisa al solicitante de que el trabajo arrancó.

3. Resolver un ticket

Al terminar el trabajo, presiona Resolver. Se abre el modal de resolución donde debes capturar:

  • Resultado: Éxito o Fallido (si no se pudo completar lo solicitado). En ambos casos el ticket queda atendido y se cierra al calificar.
  • Tipo de mantenimiento: Preventivo o Correctivo.
  • Origen: Interno o Externo (servicio contratado).
  • Tiempo invertido: minutos reales en el sitio.
  • Notas de resolución: qué se hizo, qué se reemplazó, observaciones.
  • Evidencia fotográfica: antes/después (recomendado).

4. Registrar material del almacén

Dentro del modal de resolución hay una sección para consumir materiales del Almacén Global. El sistema descuenta automáticamente del stock usando FIFO y registra la acción WAREHOUSE_MATERIAL_ADDED en la auditoría.

Si el material no existe en el almacén, repórtalo al dispatcher para que lo dé de alta antes de cerrar el ticket.
{% endif %} {% if help_all or 'maint.comments.api.internal' in user_perms %}

5. Comentar internamente

Desde la pestaña Comentarios puedes marcar un comentario como interno — lo verán solo el dispatcher, admin y otros técnicos. Útil para coordinar sin exponer al solicitante.

{% endif %} {% if help_all or 'maint.tickets.api.resolve' in user_perms %}

6. Evidencia fotográfica

Adjunta fotos en la pestaña Resolución o directamente en el modal de resolver. Límites por archivo: 3 MB, máximo 5 fotos por resolución.

Los adjuntos de tickets resueltos se borran automáticamente a los 7 días. Descarga lo que necesites antes si lo requieres como respaldo.
